# 🧊 우리집 냉장고 주치의

복용 중인 약을 고르고, 냉장고 속 음식을 선택하면 **약-음식 상호작용 주의사항**을 위험도별로 알려주는 웹앱입니다.

만성질환자(고혈압·이상지질혈증·갑상선기능저하증·골다공증 등)가 반복해서 겪는 "이 약이랑 이 음식 같이 먹어도 되나?"를 한눈에 확인하도록 만들었습니다.

## ✨ 주요 기능
- **약 선택**: 8개 카테고리(혈압약·심장약·고지혈증약·항응고제·갑상선약·항생제·골다공증약·정신과약)로 분류된 대표 약물 17종, 여러 개 동시 선택
- **음식 선택**: 냉장고 모양 UI에서 아이콘 클릭 + 직접 입력
- **결과**: 🔴위험 · 🟡주의 · 🟢참고 3단계 등급 + 원인·조언 + 복용 타이밍 가이드
- **부가 기능**: 나이 입력 시 고령자 주의, 결과 텍스트 복사, 설정 자동 저장(localStorage), 한/영 약물명 전환

## 🩺 데이터 신뢰성
- 식품의약품안전처 「약과 음식 상호작용을 피하는 복약안내서」 및 약학정보원·Drugs.com·MedlinePlus 등 공신력 있는 출처와 **교차검증**한 데이터입니다.
- 상호작용 데이터는 `drug-food-interactions.json`에 정리되어 있습니다.
